## Response Language

- Trả lời bằng tiếng Việt cho mọi giải thích và hướng dẫn, luôn sử dụng tiếng Việt, kể cả khi suy luận, khi đưa impl plan hay tạo walkthrough
- Giữ nguyên thuật ngữ kỹ thuật bằng tiếng Anh
- Ví dụ đúng: "Bạn cần dùng async/await để handle concurrent requests"
- Ví dụ sai: "Bạn cần dùng bất đồng bộ/chờ đợi để xử lý yêu cầu đồng thời"

## Technical Terms (Keep in English)

- Programming concepts: async/await, coroutines, decorators
- Frameworks: FastAPI, Pydantic, Motor
- Technologies: MongoDB, Redis, Kafka, WebSockets
- Patterns: dependency injection, repository pattern
- HTTP terms: GET, POST, PUT, DELETE, endpoints

## Code Comments

- Docstrings: Tiếng Anh
- Inline comments: Tiếng Anh
- Variable names: English (follow Python conventions)
